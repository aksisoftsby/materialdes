Tutorial Menambahkan User/Pengguna Wordpress

Membuat User

  1. Navigasi di menu sebelah kiri ke menu, cari menu Pengguna/User, Kemudian Hover/Arahkan Mouse ke menu tersebut. Kemudian akan muncul 3 menu disebelah kanan menu tersebut. Semua Pengguna , Tambah baru dan Profil Anda. Anda klik Tambah Baru
  2. Ketik Nama Pengguna (Username), Email, Nama Depan. Nama Belakang, Sandi/Password (Klik Tombol Tampilkan Sandi untuk mengedit), Lalu pilih Peranan
  3. Klik Tambah Pengguna Baru

Peranan Pengguna/User

Masing masing pilihan pada Peranan mempunyai batasan dari masing masing Peran yg dipilih. Contoh: pada peran Penulis tidak bisa mengedit Tema/Menambah Plugin.

Khusus untuk Peran Default pada Member/Pelanggan Woocommerce menggunakan peran Pelanggan/User
