Tutorial Menambahkan Produk Penjualan Barang/Jasa Wordpress Woocommerce via admin

Menambahkan Produk Penjualan

Artikel ini menggunakan menu dalam Bahasa Inggris, Kemungkinan translate menu untuk plugin woocommerce belum terinstall didalam server.

  1. Navigasi di menu sebelah kiri ke menu, cari menu Products, Kemudian Hover/Arahkan Mouse ke menu tersebut. Kemudian akan muncul menu-menu disebelah kanan menu tersebut. Anda klik Add New

  2. Product Name: Ketik Nama Product, untuk jasa ketikkan juga Jasa

    Product Description: Deskripsi product anda, Jika barang anda bisa sertakan penjelasan bahan, merk, dll. Jika jasa anda bisa tuliskan kelebihan jasa anda.

    Regular Price: Tuliskan harga pas produk anda.

  3. Pada tahap ini sebenarnya anda bisa langsung menerbitkan produk anda. Namun untuk memudahkan user mensortir produk anda jangan lupa set Category dan untuk lebih mengenalkan produk anda jangan lupa juga Upload Product Image

  4. Lalu Klik Terbitkan

Setelah anda menerbitkan produk, maka anda bisa langsung mengecek halaman Shop anda.
