Tutorial Menambahkan Halaman pada Wordpress

Membuat Halaman

Untuk semua tutorial Wordpress anda wajib login dan masuk pada Dashboard dengan url http://namawebsiteanda.domain/wp-admin

Tutorial untuk nama menu menggunakan Bahasa Indonesia

  1. Navigasi di menu sebelah kiri ke menu, cari menu Laman/Pages, Kemudian Hover/Arahkan Mouse ke menu tersebut. Kemudian akan muncul 2 menu disebelah kanan menu tersebut. Semua Laman dan Tambah Baru. Anda klik Tambah Baru
  2. Ketik Judul, dan Isi/Content dari Halaman anda.
  3. Lalu Klik Terbitkan

Shortcode

Shortcode adalah sebuah kode yang diawali [ dan diakhiri ]. Dimana kode tersebut berfungsi untuk menampilkan halaman dinamis.

Contoh jika anda pengguna Woocommerce. Anda punya halaman Cart/Keranjan belanja yang ditampilkan dengan kode [woocommerce_cart]

Shortcode ditambahkan pada halaman Content anda. Namun anda juga bisa menambahkan teks Sebelum dan sesudah kode Shortcode, contohnya: 

Data Belanja Anda  
[woocommerce_cart]
